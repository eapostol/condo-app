# Condo Management MERN Demo

This is a demo MERN application that exposes condo management reports through a role-based web portal.
It implements:

- Node.js + Express + MongoDB backend
- React + Vite + Tailwind (CDN) frontend
- JWT-based auth with bcrypt password hashing
- Google & Microsoft social login (via Passport)
- Role-based access control for manager and board views
- Swagger API docs at `/api/docs`
- Dockerfile + docker-compose for containerized deployment
